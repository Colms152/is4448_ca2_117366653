### IS4448 - Colm Sheehan (117366653) CA2 - COVID & Hero's API Application
____________________
This project is an android application built to meet the requirements of the IS4448 2021 CA2. This project interacts with an API to preform CRUD functionality to a provided API. It also presents COVID-19 statistics in a useful manner. These statistics are called from a public API.
